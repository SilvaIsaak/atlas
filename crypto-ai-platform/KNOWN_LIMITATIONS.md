# Crypto AI Platform - Known Limitations

## MVP Limitations
1. **Mock Data Only:** All data is simulated; no real backend API integration
2. **No Real-Time Updates:** No WebSocket connections; static mock data
3. **Authentication Simplified:** Login uses hardcoded mock user; no real authentication
4. **No Persistence:** No database; all state is client-side only
5. **AI Agents are Static:** No real AI agent execution; status and decisions are mock
6. **Trading is Simulated:** Orders are not executed on any real exchange
7. **Backtesting Uses Mock Results:** No real historical data or strategy execution
8. **Limited Responsiveness:** Some components may not work perfectly on very small screens

## Future Improvements
- Integrate real backend APIs
- Implement real-time WebSocket updates
- Add user authentication and authorization
- Persist data to database
- Implement real AI agent execution engine
- Connect to real crypto exchanges
- Add real historical data for backtesting
- Improve responsive design for all screen sizes
